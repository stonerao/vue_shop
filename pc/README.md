# vue 移动端单页面脚手架 

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

## 位置说明

* assets 图片
* components 组件
* routing 子路由
* mock 模拟数据
* router 路由
* styles 样式表
* utils js
* view pages
* vuex 状态管理

### 依赖项

* vue
* vue-router
* axios 
* element-ui
* mockjs
* less
